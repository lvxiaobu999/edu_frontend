# 项目规划

这是一个教育系统，先阶段有了以下模块
有user模块权限
teacher_profile 老师简介
student_profile 学生简介
role是枚举值，有 ADMIN = 'ADMIN', '管理员'、TEACHER = 'TEACHER', '老师'、STUDENT = 'STUDENT', '学生'
research_group 教研组

移除动态菜单，以及相关方法
